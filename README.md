# Stock Performance Tracker

A Python GUI application for tracking stock performance with two main features:

## Features

### 1. YTD Performance Tab
- Enter multiple stock tickers (comma-separated)
- View Year-to-Date performance for each stock
- See current price, YTD start price, and percentage change

### 2. My Portfolio Tab
- Track your actual stock holdings
- Enter ticker, number of shares, and purchase date
- Calculate real-time gains/losses
- View portfolio summary with total investment and returns

## Requirements

- Python 3.x
- yfinance
- tkinter (usually comes with Python)
- pandas

## Installation
```bash
python -m venv stock_env
source stock_env/bin/activate  # On Windows: stock_env\Scripts\activate
pip install yfinance pandas
```

## Usage
```bash
python stock_tracker.py
```
